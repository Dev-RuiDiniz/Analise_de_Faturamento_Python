Análise de Faturamento Mensal - Dashboard Interativo
Dashboard Preview

📌 Visão Geral
Este projeto é um dashboard interativo desenvolvido em Python usando Streamlit para análise de faturamento mensal. A aplicação permite:

Visualizar métricas financeiras chave

Analisar tendências temporais

Comparar desempenho entre períodos

Gerenciar e editar dados diretamente na interface

✨ Funcionalidades Principais
📈 Análise Geral
Visualização completa do histórico de dados

Gráficos interativos de faturamento, custos e lucro

Métricas resumidas e indicadores de desempenho

📅 Ano Atual
Foco nos dados do ano vigente

Comparativo mensal detalhado

Análise de sazonalidade

🔄 Comparação Anual
Análise de crescimento ano a ano

Taxas de variação percentual

Gráficos comparativos entre múltiplos anos

📝 Edição de Dados
Adição de novos registros

Edição direta dos dados existentes

Exportação/importação de datasets

Limpeza total dos dados

🛠️ Tecnologias Utilizadas
Streamlit - Framework para criação de aplicações web

Pandas - Manipulação e análise de dados

Plotly - Visualizações interativas

Python - Lógica de programação
